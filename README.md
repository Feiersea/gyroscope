MPU6050 Kernel Driver

Project Description:
The objective of thsi project is to create a driver for the MPU6050 gyroscope for the linux kernel.

I chose to use C as it is the most widely used coding language for kernel development.


Project Motivation:
My motivation for this project was to learn more about the C coding language as well as learn more about linux kernel specific development. I am starting from a backrgound of knwoing some other coding languages and then attempting to apply them to a new coding language and to a very applicable setting such as driver development.
